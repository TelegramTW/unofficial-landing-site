上個月，我們推出了 Telegra.ph，這是一個極簡主義的發布工具，允許您創建格式豐富的貼文，只需點擊一下即可將其推送到網路上。

我們試著讓 Telegraph 介面盡可能維持簡單：不需要登錄或瀏覽選單。您只要開啟 telegra.ph，寫下您的文章（包括"標題"、"您的名稱"、 "文章內容"），按「PUBLISH(發布)」，然後它就存在了。


使用機器人會更好

這種簡單性當然有它的價值。有時您想在多個裝置上編輯文章，或保持登錄狀態以避免在每個新標題下都必須寫一次您的名稱和個人資料連結。如果這是您的心聲，請來認識一下我們的新機器人：@Telegraph 。

這個簡單的機器人將幫助您在橫跨任何數量的裝置中登錄和管理您的 Telegraph 文章。此外，它可以為您提供 Telegraph 上任何文章的頁面瀏覽之統計資料。只需發送給它一個連結，然後就可以觀看這數字翻升的變化。

一個時間來收集石頭

如果您過去在不同的裝置上使用瀏覽器創建了一些 Telegraph 文章，@Telegraph 機器人可以幫忙在您的規則下聯合它們。


一個時間來登錄



然後一個時間來收集文章


只需按下"在此裝置上登錄"的按鈕，您之前在該裝置的瀏覽器中寫入的所有文章都將添加到您的 Telegraph 帳戶中。

Telegraph API

注意，Telegraph 不僅僅只為電報用戶。任何人都可以使用這個超簡單的出版工具。對於那些遭受其他通訊軟體限制而不能享受 @Telegraph 機器人所提供的舒適功能的人，我們已經建立了一個開放的 Telegraph API 。

使用 Telegraph API，任何開發人員都可以為任何其他平台創建如同 @Telegraph 一樣的機器人，甚至是獨立介面。

這裡是完整的文件，如果您有興趣的話。
